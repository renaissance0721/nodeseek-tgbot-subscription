该脚本基于TGbot：
[Flowerss源站](https://github.com/indes/flowerss-bot)
安装脚本后，会自动检测是否安装docker，没安装的会自动安装
接着输入你的TGbot token，配置完成后，跟tgbot对话，输入/sub https://www.nodeseek.com/rss.xml即可订阅
```
bash <(curl -sL https://raw.githubusercontent.com/renaissance0721/nodeseek-tgbot-subscription/refs/heads/main/install_bot.sh)
```
